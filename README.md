# 🇾🇪 موقع شات اليمن المطور

تطبيق شات متطور يعمل على الويب مع نظام متكامل للرتب والصلاحيات والإدارة.

## المتطلبات

- Node.js v16+
- MongoDB
- npm أو yarn

## التثبيت

### 1. استنساخ المشروع
\`\`\`bash
git clone <repository-url>
cd yemen-chat
\`\`\`

### 2. تثبيت المكتبات
\`\`\`bash
npm install
\`\`\`

### 3. إعداد المتغيرات البيئية
أنشئ ملف `.env` وأضف:
\`\`\`env
PORT=3000
MONGO_URI=mongodb://localhost:27017/yemen-chat
JWT_SECRET=your_secret_key_here
NODE_ENV=development
CORS_ORIGIN=*
\`\`\`

### 4. تشغيل MongoDB
تأكد من تشغيل MongoDB:
\`\`\`bash
mongod
\`\`\`

### 5. تشغيل السيرفر
#### للتطوير (مع Auto-reload):
\`\`\`bash
npm run dev
\`\`\`

#### للإنتاج:
\`\`\`bash
npm start
\`\`\`

## الوصول للموقع

افتح المتصفح وانتقل إلى: [http://localhost:3000](http://localhost:3000)

## المميزات الرئيسية

✅ دخول زوار فوري بدون تسجيل
✅ نظام رتب متدرج من زائر إلى مالك
✅ صلاحيات شاملة لكل رتبة
✅ نظام الرسائل الخاصة
✅ الأصدقاء والمتابعين
✅ نظام الإبلاغات والاعتدال الآلي
✅ نظام الرصيد والمتجر
✅ ملفات شخصية مفصلة
✅ عرض المتواجدين في الغرفة
✅ اللغة العربية بشكل كامل (RTL)

## بنية المشروع

- `server.js` - السيرفر الرئيسي
- `models/` - نماذج قاعدة البيانات
- `routes/` - المسارات (API)
- `middleware/` - البرامج الوسيطة
- `events/` - معالجات Socket.io
- `public/` - الملفات الثابتة (HTML, CSS, JS)
